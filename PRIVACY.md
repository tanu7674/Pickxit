## 2. How It Works

### Processing Your Photos

When you use SnapID Studio:

1. **Upload** — You select a photo on your device
2. **Process** — The photo is loaded into your browser's memory
3. **Analyze** (Optional) — If you provide an API key, we send the photo to Anthropic's Claude API **directly from your browser**
4. **Crop** — Your browser crops and resizes the photo
5. **Download** — You download the result to your device

**Key point:** The photo never touches a SnapID Studio server.

### API Key Usage

If you provide an Anthropic API key:

- The key stays in your browser (never sent to SnapID servers)
- Your photo is sent directly to Claude API (not through SnapID servers)
- Anthropic processes the image according to their privacy policy
- We have no way to access your photos or keys
- You control your API key and can revoke it anytime

**Your API key is like a secret handshake between your browser and Claude.**

## 4. Technical Details

### Browser Storage
The only data stored locally is what **you** choose to keep:
- Your browser's cache (optional)
- Your download folder (your choice)
- Your preferences (if using local storage)

We do not access or store this data.

### CDN & Hosting

SnapID Studio is hosted on Vercel, a Content Delivery Network (CDN).

**Vercel may collect:**
- Access logs (IP, timestamp, URL requested)
- Security metrics
- Performance analytics

**We don't access these logs.** Vercel's privacy practices are governed by their [Privacy Policy](https://vercel.com/legal/privacy-policy).

### Third-Party Services

#### Anthropic (Claude API)
- Optional, only if you provide an API key
- Governed by [Anthropic's Privacy Policy](https://anthropic.com/privacy)
- Your photos are processed according to their terms

#### QRCode.js
- QR code generation (optional, for desktop app downloads)
- No data collection
- Open source library

**No other third parties involved.**
